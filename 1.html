HTML

Hypertext Markup Language (HTML) is standard markup Language for creating web pages and web applications. Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the documents.

HTML elememts are the building blocks of HTML pages . With HTML constructs,images and other objectsn, such as interactive froms, may be embeded into the rendered page. IT provides a means to create structured documents by denoting structuarl semantics for text such as headings , paragtraphs , lists , Links , quotes and other items. HTML elements are delineated by tags, written using angle brackets.
